# 　課題

## 読み上げについて

- kaiを「けーえーあい」と読んでいる
- その他の英語もアルファベット読みになっている
- 「今日は」や「では」の「は」の読み方が「わ」になっていない
- 読み上げが一気に読み上げられているように聞こえる。音節を適切に設定できていない
- 文の間の余白が少なく、一気に読み上げられているので息継ぎがない
- 読み上げや字幕にファイル名のフルパスは合わない。ファイル名の読み上げもアルファベット読みになっている

## 配信として

- 作業に取り掛かる前に何をやるのかをしっかり伝える必要がある。プレゼン資料みたいなものがあるといい
- コーディングをするならvscodeで編集の様子を見せたい
- しばらく字幕・読み上げがない期間が続いている。何かの作業をやっていればいいが、これではリスナーがあきるかも
