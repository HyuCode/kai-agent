# kai キャラ仕様シート（多脚メカ型 kai）

> asset-production kit の `sheets/character-brief.md` を kai 用に記入したもの。
> 要件は `docs/kai/03-features/01-avatar/01-requirements.md`。**非人型のためレイヤーは眼中心に読み替える**
> （§「レイヤー読み替え」参照）。

## 基本情報

- キャラ名: kai（かい）
- 用途: AITuber 配信のアバター（PuruPuruPNGTuber で口パク・まばたき・揺れ）
- 配信画面での表示サイズ: デスクトップ隅に重ねる小〜中サイズ（後日カーソル追従）
- 想定キャンバスサイズ: 1024 × 1536（縦長。ポッド + 下部に多脚が入る）

## 見た目

- 年齢感: 無機物だが子供っぽく好奇心旺盛（人格の「ボク／みんな」に合わせる）
- モチーフ: 多脚メカ風の**多脚ロボット**、可愛くアニメタッチ
- ボディ形状: 丸みのある**ポッド型**の胴体＝頭。下部に短くて可愛い**多脚（4〜8）**が
  少し見える
- 基調色: ソフトな青緑〜水色 + 白のアクセント（多脚メカ寄り。kai 独自色に調整可）
- 「目」に相当: **大きな単眼カメラ（丸い光学レンズ）**を正面中央に 1 つ。グロッシーな
  レンズ + 中心に**発光するコア（虹彩）**
- 「口」に相当: 口は無い。**喋りはレンズコアの発光の強弱**で表す（§読み替え）
- 「髪」に相当: 頭頂の**センサー触角／アンテナ**（前方 = front、後方 = back）
- アクセサリ: 小型ライト・サブセンサー等（任意 items）
- 表情の方向性: レンズの絞り開閉（まばたき）＋コア発光（喋り）＋触角の揺れで感情表現

## ポーズと構図

- 向き: 正面固定
- 顔（＝眼ポッド）の位置: 上部中央。単眼レンズがはっきり見える
- 手や小物の有無: 手は無し。多脚は下部に控えめに
- 体の見切れ位置: 多脚の脛あたりで下辺見切れでよい
- 避けたい構図: 眼の位置がズレる／脚が主張しすぎる／斜め向き

## レイヤー読み替え（非人型 → PuruPuru の人型レイヤー）

| PuruPuru レイヤー           | kai（多脚メカ）での中身                                    |
| --------------------------- | ---------------------------------------------------------- |
| eyes-open（目開）           | 単眼レンズの**絞り開**（虹彩リングが見える＝起きている）   |
| eyes-closed（目閉/まばたき) | 単眼レンズの**シャッター閉**（横一文字に閉じる＝まばたき） |
| mouth-closed（口閉）        | レンズコアの発光 **弱**（静か・喋っていない）              |
| mouth-half（口中間）        | レンズコアの発光 **中**（喋り中）                          |
| mouth-open（口開）          | レンズコアの発光 **強**（大きく喋る）                      |
| front-hair（前髪・揺れ）    | 前方の**触角／アンテナ**（ぷるぷる揺れ）                   |
| back-hair（後ろ髪・揺れ）   | 後方の**触角／フィン**（ぷるぷる揺れ）                     |
| body.png（任意）            | ポッド型ボディ + 多脚                                      |

→ 音量駆動の口パクがそのまま「**喋るとレンズが光る**」演出になる。まばたきは
「シャッターが閉じる」。触角が揺れて無機物でも生き物っぽさが出る。
